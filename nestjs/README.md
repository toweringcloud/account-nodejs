# account-nodejs-nestjs

## user sign-up & sign-in api service using nestjs framework

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# local mode with watch
$ npm run start

# development mode
$ npm run start:dev

# production mode
$ npm run start:prd
```
